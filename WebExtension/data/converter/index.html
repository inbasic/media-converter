Audio Extractor

Extract the original audio stream from a video file. Output format will be based on audio codec. There is no setting for this conversion, just select one or more video files to extract audio tracks.

Video and Audio Combiner

Replace audio track of a video file with a new audio track. If video file already has audio, this track will be overwritten by the new one. There is no setting for this conversion, just select a video file and an audio file.

Audio Volume Adjuster

Reduce or increase audio volume of a video or audio file. Adjust the range and drop files in the box.

Video Size Scaler

Scale video screen size. This option can be used to reduce video file size by reducing its screen dimension for instance. Final width of the video will be (original size) * (m) / (d). Video height is calculated such that width to height ratio is intact. Adjust settings and drop file into the box.

Video Rotator

You can rotate a video file here. This can be helpful to watch video in a mobile device for instance. Select the rotation value and drop files into the box.





Audio Time Shifter

Sometimes, video and audio tracks are not synced. This tab can shift either audio or video file forward with a defined value to fix the syncing problem.

Video Cutting Tool

If you want to cut part of a video or audio file specify "from" time-frame and "to" time-frame and drop the file into the box. Leave "from" box empty to start from the beginning. Leave the "to" box empty to cut to the last frame. "from" and "to" can be written in seconds from zero or can be written in complete time format (hh:mm:ss.m).
